# TP6 : Gestionnaire de Tâches avec Node.js, Express.js et EJS

## Description
Ce projet implémente un gestionnaire de tâches simple avec Express.js pour le backend, EJS pour les vues dynamiques, et une API REST basique. Basé sur l'atelier guidé du PDF, avec extensions du "Travail à faire".

### Objectifs
- Configurer un serveur Express.
- Utiliser EJS pour vues dynamiques.
- Implémenter routes et API CRUD basique.
- Ajouter personnalisations : about, contact, validation, total tâches, formulaire.

## Prérequis
- Node.js.
- npm install.

## Installation
1. Clone : `git clone votre-repo.git`
2. `npm install`
3. `npm run dev` (développement) ou `npm start`.

## Routes
- GET / : Accueil personnalisé.
- GET /tasks : Liste tâches avec formulaire.
- GET /about : À propos.
- GET /contact : Contact.
- GET /profile : Profil (bonus).
- API : GET/POST /api/tasks.
